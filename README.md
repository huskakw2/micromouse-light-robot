# Micromouse-Light Robot

## Goal
This project is a small Arduino-based two-wheel autonomous robot. The goal is to learn motor control, sensors, wiring, debugging, and eventually line following or simple maze navigation

## Current Status
- Chassis assembled
- TT motors mounted
- Motor wires soldered
- Wheels attatched
- Battery pack attatched
- Waiting for TB6612FNG motor driver

## Parts
- Arduino Uno R3
- 2WD robot chassis
- 2 TT gear motors
- Wheels and caster
- Battery pack
- TBB12FNG motor driver
- Jumper wires
- Ultrasonic sensor
- Line sensor module

## Next Milestone
Wire the TB6612FNG motor driver and test one motor forward, stop, and reverse using Arduino PWM.

-Mechanical Assembly
